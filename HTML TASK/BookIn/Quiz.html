Q1. HTML은 프로그래밍 언어이다.

정답! HTML은 마크업 언어입니다.
오답! HTML은 프로그래밍 언어가 아닙니다.

Q2. CSS는 웹 페이지의 스타일을 정의한다.

정답! CSS는 웹 페이지의 시각적 스타일을 정의합니다.
오답! CSS는 웹 스타일을 정의합니다.

Q3. 자바스크립트는 서버 사이드 언어이다.

정답! 자바스크립트는 원래 클라이언트 사이드 언어입니다.
오답! 자바스크립트는 주로 클라이언트 측에서 실행됩니다.

Q4. <img> 태그는 이미지를 삽입할 때 사용한다.

정답! <img>는 이미지를 삽입하는 HTML 태그입니다.
오답! <img>는 이미지 삽입 태그입니다.

Q5. CSS에서 색상을 지정할 때 "red"와 같은 키워드를 사용할 수 있다.

정답! CSS는 색상 이름, HEX, RGB 등 다양한 방식으로 색상을 지정할 수 있습니다.
오답! CSS는 "red"와 같은 색상 키워드도 사용할 수 있습니다.
